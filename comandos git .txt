comandos git 

- git status: estado atual do repositório (arquivos que ainda não estão no repositório). 

- git add nome-arquivo: adiciona o arquivo ou modificação ao stage. 

- git commit -m "mensagem" : adiciona commit ao repositório local. 

- git log : linha do tempo do versionamento. 

- git checkout 6-digitos-head-log : volta para o versionamento desejado. 

- git checkout master/main : volta para a ultima versão do versionamento. 

- git branch nome-funcionalidade : cria uma branch da main.  

- git chekout brench-nome : mudo a brench em que estou. 

- git merge brench-segundaria : mistura a branch principal com a secundaria criada anteriormente. É necessário estar na brench main antes usando o checkout.
